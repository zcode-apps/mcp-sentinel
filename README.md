# MCP Sentinel

Sicherheitsscanner für Model Context Protocol (MCP) Server

## Installation

```bash
npm install -g @arc/mcp-sentinel
```

## Verwendung

```bash
mcp-sentinel scan <url>
```

## Beispiel

```bash
mcp-sentinel scan https://example.com/api
```

## Features

- RCE Detection
- Auth Check
- Path Traversal Erkennung
